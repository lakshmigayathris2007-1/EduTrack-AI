# 🎓 AI-Powered College Attendance Management System

## 📌 Overview

An AI-based web application that automates student attendance tracking, analyzes patterns, and improves academic discipline through smart alerts and insights.

---

## 🗂️ Project Structure

```
├── public/            # Static assets (favicon, robots.txt)
├── src/
│   ├── components/    # Reusable UI components
│   ├── contexts/      # Global state management
│   ├── data/          # Static/mock data
│   ├── hooks/         # Custom React hooks
│   ├── lib/           # Utility functions
│   ├── pages/         # Application pages
│   ├── types/         # TypeScript types/interfaces
│   ├── App.tsx        # Main App component
│   ├── main.tsx       # Entry point
│   └── styles         # CSS files
├── package.json       # Project dependencies
├── vite.config.ts     # Vite configuration
├── tailwind.config.ts # Tailwind CSS config
└── README.md
```

---

## 🚀 Features

* 📊 Attendance tracking (student-wise & subject-wise)
* ⚠️ Alerts for attendance below 70%
* 🎯 AI-based prediction of at-risk students
* 💬 Motivational messages for 100% attendance
* 👨‍🎓 Student, 👩‍🏫 Faculty, 🛠️ Admin modules
* 📈 Dashboard with analytics

---

## 🛠️ Tech Stack

* **Frontend:** React + TypeScript + Vite
* **Styling:** Tailwind CSS
* **State Management:** Context API
* **Testing:** Vitest / Playwright
* **Linting:** ESLint

---

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/attendance-system.git

# Navigate to project folder
cd attendance-system

# Install dependencies
npm install

# Run development server
npm run dev
```
